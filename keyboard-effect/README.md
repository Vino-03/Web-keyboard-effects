# 键盘按键效果演示

一个炫酷的键盘按键可视化网页应用，展示按键效果和组合键显示。

## 功能特性

- **按键可视化**：按下键盘任意按键时，屏幕中央显示对应的按键方块
- **组合键支持**：可显示组合键（如 `Ctrl+A`、`Ctrl+Shift+A`）
- **按下动画**：按键按下时方块变深色并缩小，模拟真实按键效果
- **松开效果**：所有按键松开后，方块保持1秒后淡出消失
- **星星粒子**：按键时产生6颗随机颜色的星星粒子向外扩散，带有重力效果
- **特殊按键显示**：方向键显示为箭头符号（↑↓←→），空格键显示为"Space"

## 使用方法

1. 在浏览器中打开 `index.html`
2. 按下键盘任意按键查看效果
3. 尝试组合键（如 `Ctrl+Shift`）查看组合显示

## 技术实现

- **HTML5**: 基础结构，包含一个容器div
- **CSS3**: 按键样式、淡出动画效果
- **JavaScript ES6**: 键盘事件监听、DOM操作、`KeyboardEffect` 类封装
